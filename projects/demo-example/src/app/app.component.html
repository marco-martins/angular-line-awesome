Angular Line Awesome

Simple, easy to use Angular component to manage Line Awesome icons.

How to install

Install Packages

    npm install --save line-awesome angular-line-awesome

Import the module

    //...
    import { AngularLineAwesomeModule } from 'angular-line-awesome';
    @NgModule({
    //...
      imports: [
        //...
        AngularLineAwesomeModule
      ],
      //...
    })
    export class AppModule { }

If you're using Angular CLI, add the line-awesome CSS to styles inside the angular-cli.json 

    "styles": [
        "styles.css",
        "./node_modules/line-awesome/dist/line-awesome/css/line-awesome.css"
    ],

If you're using Angular CLI 6.0.0, add the line-awesome CSS to styles inside the angular.json 

    "styles": [
        "styles.css",
        "./node_modules/line-awesome/dist/line-awesome/css/line-awesome.css"
    ],
NOTE: If using SCSS preprocessor just change the `css` for `scss` 
      "./node_modules/line-awesome/dist/line-awesome/scss/line-awesome.scss"

If you're not using the CLI, import the stylesheet to your index.html file

<link rel="stylesheet" type="text/css" 
 href="https://maxst.icons8.com/vue-static/landings/line-awesome/line-awesome/1.3.0/css/line-awesome.min.css">

Usage and options

Property Type Options Optional
icon String, IconProp Check the Line Awesome icon list.

<i class="las la-hippo"></i> becomes <la-icon icon="hippo"></la-icon>
Ignore the las- and la- part, this will be added by default. 		No
title String Free text Yes
size String xs, lg, sm, lx, 1x, 2x, 3x, 4x, 5x, 6x, 7x, 8x, 9x, 10x Yes
fixedWidth Boolean true, false Yes
rotate Number, String 90, 190, 270 Yes
flip String horizonal, vertical, both Yes
pull String left, right No
spin Boolean true, false Yes
pulse Boolean true, false Yes
border Boolean true, false Yes
inverse Boolean true, false Yes
transform String grow-NUMBER,
shrink-NUMBER,
up-NUMBER,
left-NUMBER,
right-NUMBER,
up-NUMBER,
down-NUMBER,
rotate-DEGREES,
flip-v,
flip-h

NUMBER is a number representings pixels, DEGREES is a number representings degrees, e.g. grow-1, rotate-90 		Yes

Examples

Code Icon
<la-icon icon="hippo"></la-icon>
<la-icon icon="hippo" size="2x"></la-icon>
<la-icon icon="hippo" rotate="90"></la-icon>
<la-icon icon="hippo" flip="horizontal"></la-icon>
<la-icon icon="hippo" pull="right"></la-icon>
<la-icon icon="hippo" spin="true"></la-icon>
<la-icon icon="hippo" pulse="true"></la-icon>
<la-icon icon="hippo" border="true"></la-icon>
<la-icon icon="hippo" inverse="true"></la-icon>
<la-icon icon="hippo" transform="grow-10 down-4 right-8 rotate-45 flip-v flip-h"></la-icon>

Angular bind sintaxe:

iconName: IconProp = ['lab', 'github-alt'];
iconSize: string = '2x';

<la-icon [icon]="iconName" [size]="iconSize"></la-icon>

Angular bind sintaxe:

transform: Transform = {size: 1, x: 1, y: -1, rotate: 90, flipX: true, flipY: true};

<la-icon icon="lab github" [transform]="transform"></la-icon>

Stacked icons:

square: IconProp = {prefix: 'las', iconName: 'square'};
user: IconProp = {prefix: 'las', iconName: 'user'}; 

<la-stack size="2x">
  <la-icon [icon]="square" stackItemSize="2x"></la-icon>
  <la-icon [icon]="user" stackItemSize="1x" [inverse]="true"></la-icon>
</la-stack>

Contributions

Clone repo:
git clone git@github.com:marco-martins/angular-line-awesome.git

Build:
ng build --project=angular-line-awesome

Run the project (demo-example by default):
ng serve

Run the tests:
ng test --project=angular-line-awesome
